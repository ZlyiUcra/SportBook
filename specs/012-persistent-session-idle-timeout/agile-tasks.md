### User Story 002 - An unattended, signed-in session ends itself (Priority: P1)

Progress: 6/6 tasks (100%)

Because a session can now outlive a single browser tab, a user who steps away from a signed-in
device leaves a real window open for someone else to act as them. After a period with no
activity, the user is warned with a visible countdown and given an explicit choice; if nobody
responds, the session ends on its own.
